MSC Ternary Atlas

This webpage provides an interactive visualization atlas for MSC tri-lineage differentiation analysis. Each sample is displayed as a 3D ternary plot, showing the relative osteogenic, chondrogenic, adipogenic, and differentiation-related features of cells. The atlas includes public single-cell datasets, validation materials, and prediction materials.
